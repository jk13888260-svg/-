# Unity Jump Runner 场景配置指南

由于你没有安装 Unity，我们将这款小游戏转换为了 **Web 网页游戏（H5）**！

现在你不需要下载安装任何引擎或者环境，你的游戏代码存放在当前目录下：
- `index.html` (网页结构)
- `style.css` (游戏样式与特效)
- `game.js` (核心游戏逻辑与物理引擎)

## 如何直接玩这个游戏？

你只需要找到并**双击打开**你刚发给我的这个目录里的 `index.html` 文件即可！
它会用你电脑上的默认浏览器打开，直接就可以玩。

或者，你可以在浏览器地址栏输入：
`file:///Users/mac/.gemini/antigravity/playground/exo-event/index.html`

## 游戏操作：
- **开始游戏**：点击界面的 `START GAME` 按钮。
- **跳跃躲避**：点击鼠标左键、按下键盘空格键（Space）或者向上方向键（ArrowUp）。
- **规则**：躲避从右侧飞来的红色刺客障碍物，存活越久得分越高！ 碰到障碍物或者落地游戏结束。

如果你想修改跳跃的高度、重力速度或者颜色，随时告诉我，我可以帮你调整 `game.js` 里的参数！
